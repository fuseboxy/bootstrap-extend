# bootstrap-extend
Extend JS & CSS features of Twitter Bootstrap 4.x


### Version
- 4.0


### Dependencies
- Bootstrap 4.x
- jQuery 1.9+


### CSS Features
Modal Max
Multi-Level Dropdown
Left/Right/Bottom Tab


### JS Features
Auto AJAX-Error Alert
Modal No-Cache
AJAX-Load / AJAX-Submit
Input Value Toggle
